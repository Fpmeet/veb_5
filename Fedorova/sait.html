Язык Html

HTML (от англ. HyperText Markup Language) — это язык разметки, который используется для создания структуры веб-страниц. Это фундамент любого сайта: он объясняет браузеру, где находится текст, заголовки, списки, картинки или видео.

CSS (Cascading Style Sheets) — это язык декорирования и описания внешнего вида веб-страниц, созданных с помощью HTML. Он отвечает за то, как элементы сайта (шрифты, цвета, отступы, расположение) отображаются на экране пользователя

Селекторы по классу 📚

Стили по классу в CSS — это способ применить одинаковое оформление к разным элементам на странице. В CSS имя класса задается с помощью точки, например, .button-primary, что позволяет легко управлять внешним видом множества элементов одновременно
